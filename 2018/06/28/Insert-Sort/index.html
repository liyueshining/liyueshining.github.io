Insert Sort

Insert Sort

排序算法：默认采用升序，两个遍历，第一个是全范围的遍历，
第二个是首元素和当前元素之前的小范围遍历，使元素在该小范围内保持有序，
如果第二个遍历中的当前元素大于第一个遍历中的当前元素，
则将该元素及后面的元素整体向后移动一位，并小的元素插入到第二个遍历中当前元素的位置。

时间复杂度：O(N^2)

空间复杂度：O(1)

稳定性： 稳定
